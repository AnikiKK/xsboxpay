API对接文档

一、发起付款接口 ${api}/xsbox-pay/api/pay/subOrder
SDK demo 下载； java-sdk.zip    php-sdk.zip    点击文件即可下载
传参方式：Post

使用方法：用表单post的方式

请求参数：
# 参数名 含义 类型 说明
1 appId 应用ID String 必填。进入商户后台 -> 运营管理 -> 商户应用查看
2 appSecret 应用私钥 String 必填。进入商户后台 -> 运营管理 -> 商户应用查看
3 mchNo 平台商户号 String 必填。就是登录的商户号
4 mchOrderNo 商户订单号 String 必填。商户唯一订单。
7 amount 支付金额 long 必填。支付金额 单位：分
8 subject 商品名称 String 必填。您的商品名称，用来显示在后台的订单名称。
9 body 商品描述 String 简单描述商品
5 payWay 支付方式 String 必填。目前只支持 -> 微信：WEIXIN, 支付宝：ALIPAY
10 notifyUrl 异步通知地址 String 异步通知地址
二、付款成功回调通知

用户付款成功后，我们会向您在发起付款接口传入的 notify_url 网址发送通知。您的服务器只要返回大写字符串“SUCCESS”（不包括引号），就表示回调已收到。如果返回不是SUCCESS，我们会再尝试回调5次，每次间隔1 5 10 15 20分钟。

传参方式：GET
参数内容：
# 参数名 含义 类型 说明
1 appId 应用ID String 应用ID
3 mchNo 平台商户号 String 就是登录的商户号
4 mchOrderNo 商户订单号 String 商户唯一订单
9 orderId 支付订单ID String 支付订单ID
7 amount 支付金额 long 支付金额 单位：分
8 orderStatus 支付状态 String 成功 SUCCESS 非 SUCCESS 就是不成功
9 msg 支付信息 String 支付成功 或 支付失败等等
